# Motion Profiles

Pick one profile that matches the page.

## Quiet Editorial

Use for:

- minimal landing pages
- documentation-like experiences
- premium editorial layouts

Behavior:

- blur-and-rise text reveals
- small hover lifts
- very light staggered entry
- rare looping elements

## Product Pulse

Use for:

- SaaS
- AI products
- product showcases

Behavior:

- staged section entry
- framed product surfaces with subtle parallax
- hover states with internal movement
- one or two alive surfaces per viewport

## Technical Motion

Use for:

- infrastructure
- dashboards
- developer tools

Behavior:

- precise, efficient transitions
- tighter durations
- less decorative easing
- stronger state feedback than ambient animation

## Cinematic Narrative

Use for:

- launches
- portfolio pieces
- scroll-led storytelling

Behavior:

- pinned sequences
- coordinated section reveals
- layered depth
- stronger transitions, but only around the core narrative moments

## Intensity Mapping

Match the profile to the requested intensity:

- `1-3`: only essential entry and hover motion
- `4-6`: standard premium motion baseline
- `7-8`: stronger choreography, sticky patterns, shared transitions
- `9-10`: cinematic treatment only if the content and performance budget justify it

# Review Rubric

Run this before final output.

## Sequence

Check:

- Is the entry order clear?
- Do the most important elements arrive first?
- Does the animation support reading instead of interrupting it?

## Interaction

Check:

- Do buttons and links feel tactile?
- Are hover and press states subtle but obvious?
- Is focus-visible still clear?

## Scroll Behavior

Check:

- If sticky or pinned behavior exists, does it clarify the story?
- Are scroll-driven changes limited to meaningful moments?
- Has the mobile version been simplified where needed?

## Restraint

Check:

- Is motion concentrated in a few important places?
- Are infinite loops rare and justified?
- Does the page still feel calm enough to read?

## Performance

Check:

- Are only safe properties being animated?
- Are expensive effects isolated or minimized?
- Would this still feel smooth on a mid-range device?

## Failure Signs

Revise if the output feels like:

- animation added on top of a weak layout
- every card moving for no reason
- sticky effects used as gimmicks
- hover states that feel game-like instead of premium

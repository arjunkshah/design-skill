# Prompt Recipe

Use these when the model needs clearer motion direction.

## Short Prompt

Use `$animation-director` to add restrained, high-quality motion that clarifies hierarchy, reinforces interaction, and stays performant.

## Combined Prompt

Use `$layout-director $animation-director`.

Requirements:

- choose a strong layout system first
- pick one motion profile that matches the product
- keep animation focused on entry order, hover feedback, sticky storytelling, or product framing
- preserve mobile clarity and performance

## Structured Prompt

Use `$animation-director`.

Motion goals:

- what needs to feel alive
- what should stay quiet
- whether the page needs sticky storytelling, hover polish, or active product surfaces

Constraints:

- target stack
- allowed libraries
- performance sensitivity
- preferred motion intensity

Quality bar:

- transform and opacity first
- no decorative animation spam
- clear CTA and affordance feedback
- mobile-safe simplification

## Refinement Prompt

Keep the same layout and visual identity, but upgrade the motion system. Increase sequence clarity, improve hover tactility, and concentrate animation in fewer, stronger moments.

# Variation Matrix

This file exists to keep the skill flexible without making it random.

Choose one option from each axis based on the brief.

## 1. Structure

Choose from [layout_archetypes.md](layout_archetypes.md).

The structure is the main organizing logic of the page.

## 2. Typography Mode

Pick one:

- **Serif-led**: for editorial, luxurious, reflective, or premium positioning
- **Grotesk-led**: for product clarity, modernity, confidence
- **Compressed display**: for bold campaigns or energetic launches
- **Quiet functional**: for dense expert products where clarity matters most

## 3. Density

Pick one:

- **Airy**: fewer elements, more whitespace, larger gestures
- **Balanced**: moderate information with calm pacing
- **Dense**: more proof and detail, still strongly ordered

## 4. Section Rhythm

Pick one:

- **Calm blocks**: long sections with fewer interrupts
- **Alternating compression**: broad moments followed by tighter proof clusters
- **Staggered beats**: modular sections with varied card sizes and pacing
- **Narrative progression**: each section advances the story in sequence

## 5. Motion Profile

Pick one:

- **Minimal**: only essential reveal and hover behavior
- **Measured**: stronger staged entry motion and subtle depth
- **Expressive**: more visible but still controlled movement for campaign pages

## 6. Viewport Behavior

Pick one:

- **Static flow**: normal stacked sections with no pinned behavior
- **Sticky story**: one preview remains pinned while adjacent content advances
- **Pinned media**: a product or visual frame stays fixed while supporting text scrolls
- **Rail motion**: horizontal or staggered module movement inside a larger section
- **Layered stack**: overlapping cards or sections that build depth as the user scrolls

## Combining Axes

Examples:

- Editorial Split + Serif-led + Airy + Calm blocks + Minimal + Static flow
- Asymmetric Bento + Grotesk-led + Balanced + Staggered beats + Measured + Rail motion
- Product Frame + Quiet functional + Dense + Alternating compression + Minimal + Sticky story

Do not select combinations that fight the brief.

## Rule

If a page feels repetitive, switch one or two axes deliberately. Do not rewrite the page with random cosmetics.

# Agent Scope

Use only these files as skill instructions:

- `skills/design-skill/SKILL.md`
- `skills/animation-director/SKILL.md`

Treat everything else in this repository as product, packaging, or marketing material.

In particular:

- `index.html` is the landing page for the skill repo
- `examples/` contains showcase outputs and screenshots
- `public/` contains landing-page assets
- `README.md` is user-facing documentation

Do not treat the landing page, repo copy, or marketing assets as design instructions for user work unless the user explicitly asks to edit the landing page itself.

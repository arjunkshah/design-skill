How The Skill Thinks

It makes layout decisions in layers, not vibes alone.

The point is to keep freedom where design needs it and add hard guardrails where coding agents usually fail.

Step 1

Read the brief like a creative director

Extract audience, product maturity, brand tone, proof density, interaction weight, and mobile constraints before drawing anything.

Step 2

Select variation axes

Choose a layout archetype, type contrast mode, spacing scale, CTA hierarchy, and section rhythm that fit the brief.

Step 3

Run a ruthless quality check

Reject layouts that feel cramped, overfilled, too symmetrical, too obvious, or too animated. Keep the output usable and premium.

Variation Matrix

The skill keeps outputs fresh by changing the right variables, not by improvising blindly.

Axis 1

Structure

Pick the primary page system first so the composition has a clear backbone.

Axis 2

Typography Mode

Decide whether the page is serif-led, grotesk-led, compressed, or quieter and functional.

Axis 3

Density

Control how much information each viewport carries instead of defaulting to evenly distributed content.

Axis 4

Section Rhythm

Alternate expansion, compression, and proof moments so the page has momentum.

Axis 5

Motion Profile

Match motion intensity to the brand and use it only to clarify sequence or depth.
